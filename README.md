# VNT Aura Skin & Wellness

A premium, dynamic web application built for the VNT Aura Skin & Wellness clinic.

## Features
- **Dynamic Content:** Fully functional PHP/SQLite backend allowing the founder to manage site content directly from an admin dashboard.
- **Booking System:** Integrated consultation booking form that saves requests to a secure database and dispatches email notifications.
- **Media Manager:** Built-in dashboard to swap out website images (Hero, About, Treatments, Founder portraits) without touching code.
- **Responsive Design:** A custom, elegant front-end crafted with vanilla HTML/CSS, perfectly suited for mobile, tablet, and desktop views.
- **Premium Aesthetics:** Employs a luxurious color palette (Dark Olive & Gold) and beautiful typography to match the brand identity.

## Tech Stack
- Frontend: HTML5, CSS3, JavaScript (Vanilla)
- Backend: PHP 8+
- Database: SQLite

## Setup Instructions
1. Clone this repository into your local web server environment (e.g., WAMP, XAMPP).
2. Ensure PHP is running.
3. Access `index.php` through your localhost URL (e.g., `http://localhost/VNTAuraSkin/index.php`).
4. Access the admin dashboard at `login.php` (Default username: `vnt_admin`, Password: `password123`).
